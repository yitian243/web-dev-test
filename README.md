# Web Developer Technical Test

This repository contains the completed technical test for the Web Developer role.  
It includes three tasks, organized in folders **A**, **B**, and **C**. Each folder is self-contained and can be run independently.

---
## Getting Started

1. **Clone the repository:**

```bash
git clone https://github.com/yitian243/web-dev-test.git
```
2. **Change directory to the project folder:**
```bash
cd web-dev-test
```
3. **Then follow the setup and run instructions for each folder below.**
---
## Folder A – Task A

**Language:** C++

**Description:**  
Contains the solution for Task A.  

**Setup & Run:**  

1. Make sure you have g++ installed.

2. Open a terminal in the A folder.

3. Compile the program:

```bash
g++ a.cpp -o a_solution
```
4. Run the program:
```bash
./a_solution
```

## Folder B – Task B

**Language:** C++

**Description:**  
Contains the solution for Task B.  

**Setup & Run:**  

1. Make sure you have g++ installed.

2. Open a terminal in the B folder.

3. Compile the program:

```bash
g++ b.cpp -o b_solution
```
4. Run the program:
```bash

```

## Folder C – Task C

**Technology**: HTML, CSS, JavaScript

**Description**:
Frontend-only solution for Task C. All functionality is handled on the frontend; no backend or build tools are required.

**Setup & Run**:

1. Open the C folder.

2. Open the index.html file in a web browser (Chrome, Edge, Firefox, etc.).

3. The login page / frontend app should appear and work immediately.

Notes:

- No installation is required.

- All functionality runs in the browser.

- Make sure all HTML, CSS, and JS files (index.html, style.css, script.js) are in the same folder.
